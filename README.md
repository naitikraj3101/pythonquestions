# Python Questions

This repository contains a collection of Python programming problems and their solutions. Each question is implemented in a separate Python file (q1.py to q20.py).

## Questions

1. **q1.py**: Python program to add two numbers
2. **q2.py**: Python program to solve quadratic equation
3. **q3.py**: Python program to swap two variables
4. **q4.py**: Program to check if a number is positive or negative or 0
5. **q5.py**: Program to find the largest among three numbers
6. **q6.py**: Program to check prime number
7. **q7.py**: Program to find the factorial of a number
8. **q8.py**: Fibonacci series (iterative)
9. **q9.py**: Armstrong numbers in a range
10. **q10.py**: Sum of first n natural numbers
11. **q11.py**: Powers of 2
12. **q12.py**: Numbers divisible by a given divisor
13. **q13.py**: Decimal to binary, octal, and hexadecimal conversion
14. **q14.py**: GCD of two numbers
15. **q15.py**: LCM of two numbers
16. **q16.py**: Simple calculator (+, -, *, /)
17. **q17.py**: Fibonacci series (recursive)
18. **q18.py**: Basic class example
19. **q19.py**: Inheritance example
20. **q20.py**: Class with private attributes

## How to Run

To run any program, use the following command in your terminal:

```
python q[number].py
```

For example:
```
python q1.py
```

Most programs take input from the user via the console.
